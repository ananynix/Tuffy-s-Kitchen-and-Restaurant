
# Tuffy’s Kitchen and Restaurant

This project is a comprehensive **full-stack food delivery web application** developed as part of the Web Programming course for Semester IV. It features an intuitive frontend for users to browse, select, and order dishes, supported by a robust backend for authentication and order processing.

---

## Core Functionalities
The application replicates the features of leading food delivery platforms, including:
* **User Authentication:** Secure login and registration for customers and administrators.
* **Restaurant & Menu Browsing:** Dynamic display of available restaurants with filtering by cuisine and ratings.
* **Cart Management:** Users can add or remove items, view total prices, and proceed to checkout.
* **Order Tracking:** Real-time updates on status from preparation to delivery.
* **Admin Dashboard:** Interface for administrators to manage restaurants, menus, and orders.

---

## Tech Stack
### Frontend Technologies
* **React.js:** Core library for building the user interface.
* **React Router:** For seamless dynamic routing and navigation between pages.
* **Axios:** Handles HTTP requests and RESTful API communication with the backend.
* **Bootstrap:** Provides pre-designed components for faster UI development.
* **CSS Flexbox & Grid:** Ensures a responsive design compatible across various devices.
* **Context API & Hooks:** Utilized for effective application state management.

### External Integrations
* **Stripe:** Integrated for handling credit/debit card payments during checkout.
* **Font Awesome:** Supplies a wide range of icons to enhance visual appeal.

---

## Application Pages
1. **Home Page:** Features a search bar, category filters, and featured restaurants.
2. **Restaurant Page:** Displays selected restaurant menus, ratings, and reviews.
3. **Cart Page:** Lists selected items, quantities, and total price with modification options.
4. **Checkout Page:** Collects delivery details and provides payment methods (COD or Stripe).
5. **My Orders:** Provides users with a history of past orders and tracking options.
6. **Admin Panel:** Allows for adding/listing food items and managing incoming orders.

---

## Conclusion
The project successfully delivers a responsive and interactive user experience by leveraging the React.js ecosystem. The structured integration between the frontend and backend provides a solid foundation for future scalability and feature enhancements.
